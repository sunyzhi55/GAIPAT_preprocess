
## 你必须把数据预处理成统一格式（非常关键）
### 最终要输出的格式（每一行=一帧）
```
subject_id, frame, gaze_x, gaze_y, target_x, target_y, error_label
```

### 每一列含义
- `subject_id`：人编号（用于跨对象实验）
- `frame`：时间帧（保证时序）
- `gaze_x, gaze_y`：**校准后的眼动坐标**
- `target_x, target_y`：积木/目标坐标
- `error_label`：**0=专注，1=分心/失误**（真实标签）

# 2. 代码怎么构建？每一步做什么、目的是什么？
我给你**完整代码流水线 + 每一步作用**，照着写就能跑通。

---

## 步骤1：读取原始数据 → 提取需要的字段
**代码作用**
把原始复杂数据，只留下你需要的5个东西：
人ID、眼动、目标、时间、是否失误

**目的**
去掉无用数据，减少后续计算量。

---

## 步骤2：空间校准（仿射变换）
**代码作用**
把眼动坐标从「摄像头坐标系」映射到「工作台坐标系」

**公式**
```
gaze_real = affine_matrix @ gaze_camera
```

**目的**
消除：身高、坐姿、摄像头角度带来的个体差异
→ **让所有人在同一个空间里比较**

---

## 步骤3：计算注视-目标偏差序列
**代码作用**
每一帧算一个偏差：
```
offset_x = gaze_x - target_x
offset_y = gaze_y - target_y
```
把一段视频变成一串**偏差序列**：
```
[ (dx1,dy1), (dx2,dy2), (dx3,dy3), ... ]
```

**目的**
这就是**专注力动力学轨迹**
专注的人轨迹稳定 → 偏差小
分心的人轨迹乱 → 偏差大

---

## 步骤4：构建标准专注模板
**代码作用**
把所有**成功任务**的偏差序列对齐、求平均
→ 得到一条**标准专注曲线**

**目的**
作为“标准答案”，用来对比每个人像不像专注。

---

## 步骤5：用 DTW 计算 ADF 专注力指纹
**代码作用**
计算**你的偏差序列**和**标准模板**的 DTW 距离
```
adf_score = dtw(your_sequence, template_sequence)
```

**目的**
ADF 分数越小 → 越专注
ADF 分数越大 → 越分心

---

## 步骤6：轻量级特征提取网络（0.08M）
**代码作用**
把时序偏差序列 → 压缩成一段低维特征向量

**目的**
小、快、边缘可运行，不依赖大模型。

---

## 步骤7：GRL 对抗解耦（去掉个人特征）
**代码作用**
一个分支判断专注力
一个分支判断“这个人是谁”
用 GRL 让**身份判别器失效**

**目的**
强迫模型只看专注力，不看人脸、习惯、动作风格
→ **跨对象通用**

---

## 步骤8：小样本学习（5-shot、10-shot）
**代码作用**
给模型看某个人5个样本 → 立刻适配这个人

**目的**
解决高危岗位**数据少、不能大量标注**的问题。

---

## 步骤9：LOSO 跨对象测试
**代码作用**
训练：19个人
测试：第20个人

**目的**
验证模型**真的能跨人通用**，不是死记硬背。

---

## 步骤10：跨数据集泛化（GAIPAT → GAZELOAD）
**代码作用**
在乐高上训练 → 直接在机器人装配任务上测试

**目的**
证明你的方法**通用、不依赖任务、不依赖场景**。
